Generate Post Grid with Divi builder from any post type.

== Description ==

This plugin adds a new Divi Module ready to create post grids from any post types. Select post type, taxonomy, order and more, all from Divi Builder and generate grids for your blog post, team members, movies, product, courses or any other post type with just a few clicks.

You can stylize your grid directly using Divi design options.

= Features =
* Simple & Intuitive
* Divi builder support
* Custom post type
* Order
* Posts limit
* Post offset

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Advanced Custom Types for Divi"
3. **Activate** Advanced Custom Types for Divi from your Plugins page
4. **Go** to Divi Builder and select ACT Post Grid module.
5. **Select** post type, layout and customize your first post grid.
